Introducción

En este curso aprenderás a crear aplicaciones móviles híbridas con el framework ionicframework, seleccionamos este framework porque además de ser bonito visualmente usa como arquitectura angularJS, el cual veremos más adelante tiene muchas ventajas. Del lado de back end usaremos Java con el framework Spring y como framework de mapeo utilizaremos Hibernate.

Creación de una aplicación.

Una vez instalado node.js para el sistema operativo en el que vayas a trabajar el primer paso es instalar en la maquina ionic con ayuda de node.js, para ello vas a ejecutar en una terminal la siguiente línea.

$ npm install -g cordova ionic

Posteriormente vas a crear una aplicación, para ello vas a espedir la siguiente línea de comandos

$ ionic start MiAplicacion blank

La linea anterior crea una aplicación en blanco, MiAplicacion es el nombre con que se generará una carpeta donde estarán contenidos todos los archivos y recursos de tu aplicación, puedes sustituir la opción blank con tabs o sidemenu y se generarán plantillas ya más completas.

Una vez creado el proyecto vas a observar una serie de carpetas, este proyecto es para trabajarlo directamente con nodejs , sin embargo lo vamos a poner en producción con NetBeans y del portal oficial de PhoneGap. Arquirectura de una aplicación Ionic

Como en el proyecto anterior creamos una nueva aplicación en este caso con la opción blank, la llamaremos AplicacionEventos Procedemos como en el ejercicio anterior, de la carpeta que se genere, buscamos la carpeta wwww, y creamos un nuevo proyecto html5 en Netbeans, borramos el archivo index.html que se genera y en la carpeta raíz pegamos el contenido de carpetas y archivos que están dentro de la carpeta wwww. Nota: NO PEGUES LA CARPETA wwww, SE PEGA EL CONTENIDO DE ELLA.

Ionic genera la plantilla del proyecto junto con todo lo necesario para que puedas utilizar angularJS, En el archivo app.js vamos a agregar un controller que se llamara 'ctrlPrincipal' como se ilustra en el siguiente código.

angular.module('starter', ['ionic'])

.run(function($ionicPlatform) {
  $ionicPlatform.ready(function() {

    if(window.cordova && window.cordova.plugins.Keyboard) {
      cordova.plugins.Keyboard.hideKeyboardAccessoryBar(true);
    }
    if(window.StatusBar) {
      StatusBar.styleDefault();
    }
  });
}).controller('ctrlPrincipal',function($scope, $http){
    $scope.mensaje='Bienvenido a angular con ionic!!';
})

Ahora vas a editar el archivo index.html y vamos a adaptarlo para invocar el controlador que ya hemos introducido en el archivo app.js. Modifica el archivo para que quede de la siguiente manera.

  <body ng-app="starter">

    <ion-pane>
        <ion-header-bar class="bar-stable" ng-controller="ctrlPrincipal">
        <h1 class="title">{{mensaje}}</h1>
      </ion-header-bar>
      <ion-content>
      </ion-content>
    </ion-pane>
  </body>

Angular maneja el patron de diseño MVC (Model View Controller), en este patrón el bloque central es el controller, en el se comunican la capa vista (en este caso el tu pagina html y als componentes que existen en el DOM) y las componentes del Model, las cuales se relacionan a los datos que van a enviarse o recibirse del Front End. En la capa controller en angular podemos comunicarnos con el model y la vista mediante la variable $scope que recibe como argumento la función de controller respectivo. Con esta variable generas cualquier tipo de atributos que serán comunicados y usados en tu DOM. Asi en este ejercicio creamos la variable $scope.mensaje, la cual se usa en el DOM con la sintaxis {{mensaje}} En la barra de la aplicación hemos utilizado {{mensaje}} donde puedes observar que toma el valor proporcioando en el controller.
